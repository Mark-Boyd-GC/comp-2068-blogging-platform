1) UserModel - defines our user document
    - first name
    - last name
    - email
    - password
2) UserController - defines our actions
    - new (register)
    - create
3) User Views - define the visual pages
    - new (register)
4) User Routes - defines the path to the controller action
    - GET route to new(_new)
    - route to create
5) Register the routes with the route composer
